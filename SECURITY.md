# 🔐 Security Policy

## Supported Versions

This project is in early development.  
Only the latest version is actively maintained.

| Version | Supported |
|--------|----------|
| 17.05.2026.1 | Yes |

---

## 🔎 Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

### Please include:
- Description of the issue
- Steps to reproduce
- Possible impact

---

## 🚫 Do NOT:

- Publicly disclose security vulnerabilities before reporting
- Attempt to exploit issues in production or shared environments

---

## 📬 Where to Report

Open a GitHub Issue or contact the maintainer privately if available.

---

## 🔒 Security Notes

Since this is a local SQLite-based project:
- Data is stored locally on your machine
- No network communication occurs by default
- No authentication system is implemented yet

Future versions may include:
- API security layers
- Input validation improvements
- Rate limiting for abuse prevention
